Install Docker (Points)

Search “Docker Desktop Windows”.

Click Download.

Open the installer.

Click Install.

Restart PC.

Open Docker Desktop.

✅ Install Node.js + npm (Points)

Search “Node.js download”.

Download LTS version.

Open installer → Install.

Check:

node -v
npm -v

✅ Test

Open CMD.

Run:

docker run hello-world
