Go to Node.js website

Open your browser

Visit nodejs.org

2. Download Node.js

Click LTS (Recommended)

Download starts

3. Install Node.js

Open the downloaded file

Click Next

Accept terms

Click Next → Install

Wait

Click Finish

4. Check installation

Open Command Prompt / Terminal
Type:

node -v → shows Node version
